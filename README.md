# Banking-Fraud-Detection-Risk-Analytics-Dashboard
Project Overview
The Banking Fraud Detection & Risk Analytics Dashboard is an interactive Power BI solution designed to monitor banking transactions, identify fraudulent activities, analyze risk indicators, and support fraud investigation processes. The dashboard provides a comprehensive view of transaction behavior, fraud patterns, authentication risks, and high-risk transactions, enabling data-driven decision-making for risk management teams.
Objectives
Detect and monitor fraudulent banking transactions.
Analyze transaction risk using multiple risk indicators.
Identify high-risk channels and authentication methods.
Monitor fraud trends across different time periods.
Support fraud investigation through detailed transaction analysis.
Provide actionable insights for fraud prevention and risk mitigation.

Key Insights
Fraudulent transactions account for 12.51% of all transactions.
Mobile App transactions show the highest fraud volume among all payment channels.
Fraud activity is most prominent during evening hours.
OTP-based authentication records the highest number of fraudulent transactions.
High-risk transactions are identified using device risk scores, anomaly detection scores, and behavioral indicators.
International transactions and suspicious IP activities contribute significantly to transaction risk assessment.
